# スマホで使う手順

## 画面確認

Mac とスマホを同じ Wi-Fi に接続して、スマホのブラウザで開きます。

```text
http://192.168.11.24:5176
```

このURLはHTTPなので、スマホでは画面表示とデモ動作の確認用です。

## スマホの実カメラで使う条件

iPhone Safari / Android Chrome でカメラを使うには、ページをHTTPSで開く必要があります。
LANの `http://192.168...` では、ブラウザがカメラAPIをブロックします。

実カメラで使う場合は、このフォルダの静的ファイルをHTTPSで配信してください。

- `index.html`
- `styles.css`
- `app.js`

GitHub Pages、Netlify、Vercel、Cloudflare Pages などのHTTPSホスティングに置けば、スマホの背面カメラで動きます。

## 使い方

1. HTTPSのURLをスマホで開く
2. 「カメラ開始」を押す
3. カメラ権限を許可する
4. 2本のゲートが道路を横切るようにスマホを固定する
5. 実際のゲート間距離を「ゲート間距離」に入力する

測定中はスマホを動かさないほど速度推定が安定します。
